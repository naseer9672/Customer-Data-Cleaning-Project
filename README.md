# Customer Data Cleaning Project

## Project Overview

This project focuses on cleaning and preparing customer marketing data for analysis using Microsoft Excel. The dataset contained missing values, duplicate records, inconsistent formatting, and raw customer information that required preprocessing before analysis.

## Data Cleaning Tasks Performed

* Removed missing values from important columns
* Removed duplicate customer records
* Standardized text formatting using Excel formulas
* Created a new Age column from Year_Birth
* Identified and removed unrealistic values
* Organized and formatted the dataset professionally

## Tools Used

* Microsoft Excel
* GitHub

## Outcome

The raw dataset was transformed into a clean and structured format suitable for business analysis and future data visualization projects.
